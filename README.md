# 県市切替マクロ

施工計画書・施工体制台帳ブック向けに、入力シートで「県」「市」を選ぶだけで対象シートを自動で表示／非表示にする VBA マクロです。

---

## はじめに（まずはここだけ）

**最短 4 ステップでセットアップ完了します。**

1. Excel で対象ブックを開いて **`Alt + F11`** を押す（VBA 画面が開きます）
2. メニュー **`ファイル → ファイルのインポート`** で **`県市切替マクロ.bas`** を選ぶ
3. Excel 画面に戻って **`Alt + F8`** → **`初期設定`** を選んで **`実行`**
4. 入力シート (`施工計画書（はじめに記入）`) の **`D3` セル**の ▼ から **「県」または「市」** を選ぶ
   （`Delete` キーで空欄に戻すと県・市 両方表示）

> うまくいかないときは下の **「トラブル対応」** を、提出前には **`設定チェック`** マクロ（**「提出前・配布前チェック」** 参照）を 1 回実行してください。

---

## できること

- 入力シート **施工計画書（はじめに記入）** の **D3** セルに「**県**」または「**市**」を選ぶと、
  - 県土木向けシート群 ↔ 市役所向けシート群 を一括で切替（片方を表示／もう片方を非表示）
- D3 にはドロップダウン（▼）が自動で付き、誤入力を防止
- 「全シートが非表示になる」操作は自動で中断
- 入力シート自身は誤って非表示にならないよう常時保護

---

## 対象ブックの前提

| 項目 | 値 |
| --- | --- |
| 対象シート名 | `施工計画書（はじめに記入）` |
| 入力セル | `D3` |
| 入力値 | `県` / `市` / 空欄（空欄=県・市 両方表示） |
| 対象ブックの保存形式 | **マクロ有効ブック**（`.xls` または `.xlsm`） |

> シート名は **タブに表示されている名前と完全一致**している必要があります（半角／全角・スペース有無に注意）。

## 導入手順（詳しい説明）

> 上の「**はじめに（まずはここだけ）**」と同じ手順を、画面操作を丁寧に追って説明します。

### ① マクロを取り込む

1. 対象 Excel ブックを開く
2. `Alt + F11` で VBE（Visual Basic Editor）を開く
3. メニュー `ファイル → ファイルのインポート` を選択
4. `県市切替マクロ.bas` を選択して開く
5. プロジェクトツリーに `県市切替` モジュールが追加されることを確認

### ② 初期設定を実行

1. Excel に戻る
2. `Alt + F8` でマクロ一覧を開く
3. `初期設定` を選択 → 実行
4. 完了メッセージが表示されたら OK

これで D3 セルに ▼ が表示され、`県`／`市` を選ぶだけで切替が動作します。

## 使い方

1. `施工計画書（はじめに記入）` シートを開く
2. `D3` の ▼ から **県** または **市** を選択（**空欄に戻すと県・市 両方表示**）
3. 対応するシートが自動で表示／非表示に切り替わる
4. 入力シートの **`H3:K30`** に「提出書類ナビ」（下記参照）が自動再生成される

> **提出先未選択に戻したい場合**: D3 を選んで `Delete` キーで空欄にしてください。県シート・市シートの全てが表示され、ナビは両方のリンクが並びます。

---

## 提出書類ナビ

D3 で「県」「市」を切り替えるたびに、入力シートの **`H3:K30`** 範囲にジャンプ用一覧が自動で再生成されます。

### 表示される内容
- 見出し: `提出書類ナビ`
- 現在の提出先:
  - `県` のとき → `県土木`
  - `市` のとき → `市役所`
  - **空欄のとき** → `未選択（県・市 両方表示）`、続けて `【県土木】` ヘッダ → 県シート群 → `【市役所】` ヘッダ → 市シート群
- 対象シート名のハイパーリンク（クリックでそのシートにジャンプ）

### 動作の特徴
- D3 を変更するたびに**毎回作り直し**（古いリンクは消える）
- `県シート一覧()` / `市シート一覧()` で**実在するシートだけ**がリンクとして並ぶ
- 不在のシートはここには出ません（不在は `設定チェック` で検出する想定）
- 想定外の値の場合はナビ領域がクリアされるだけ

### ⚠ 重要: H3:K30 はナビ専用領域

**入力シートの `H3:K30` は提出書類ナビ専用領域**です。「県」「市」を切り替えるたびに内容が**消去・上書き**されます。

- 手書き値・別ハイパーリンク・書式付きセル等を**この範囲に置かないでください**
- `初期設定` 実行時に H3:K30 に既存データが見つかると、上書き確認ダイアログが出ます（既存のナビは除外）
- 既に値がある場合は別の場所へ退避してから `初期設定` を実行してください

## 提出前・配布前チェック（推奨）

完成版を提出・配布する前に **`設定チェック`** マクロを 1 回実行してください。シート設定の不整合を一括で洗い出せます。

### 実行手順
1. `Alt + F8` → `設定チェック` を選択 → 実行
2. 「設定チェック完了。問題は見つかりませんでした。」が出れば OK
3. 問題が出たら MsgBox にリスト表示されるので、各項目を直す

### チェック項目
- 入力シート (`施工計画書（はじめに記入）`) が存在し、**表示状態**になっているか
- 入力セル (`D3`) の現在値が **空欄 / 県 / 市** のいずれかか（空欄は両方表示として正常扱い）
- `県シート一覧()` の全シートが**実在**するか
- `市シート一覧()` の全シートが**実在**するか
- 県リストと市リストに**重複するシート名**がないか
- 入力シートに **`Worksheet_Change` イベント**が存在し、かつ `県市シート切替` を呼び出しているか（VBA プロジェクトへの信頼アクセスが OFF だと「未確認」扱いで警告）

> シート名のタイポや (2) の付け忘れなど、目視では気付きにくい不整合をまとめて検出します。

## トラブル対応

### 「シートが見つかりません」が出る

- 入力シートの**タブ名がマクロの想定と違う**ことが原因
- 対処法 A: シート名を `施工計画書（はじめに記入）` に揃える
- 対処法 B: `県市切替マクロ.bas` を編集して `Private Const 入力シート名 As String = "..."` を実際のシート名に変更し、ブックを保存し直す

### 「セットアップ未完了」が出る

メッセージ通り、Excel の以下の設定を 1 つ有効にしてからもう一度 `初期設定` を実行してください。

1. Excel メニュー `ファイル → オプション → トラストセンター`
2. `トラストセンターの設定 → マクロの設定`
3. `VBA プロジェクト オブジェクト モデルへのアクセスを信頼する` にチェック
4. OK で閉じて、もう一度 `初期設定` を実行

それでも解決しない場合は、表示されたメッセージ末尾のコードをコピーし、下記「手動貼付」を実施してください（メッセージ最後の `（技術情報: Err ...）` 部分は技術者向けの参考情報なので無視して構いません）。

### Worksheet_Change を手動で貼り付ける

1. `Alt + F11` で VBE を開く
2. プロジェクトツリーから `Sheet1 (施工計画書（はじめに記入）)` をダブルクリック
3. 右側のコード窓に以下を貼り付け
4. ブックを保存

```vb
Private Sub Worksheet_Change(ByVal Target As Range)
    If Intersect(Target, Me.Range("D3")) Is Nothing Then Exit Sub
    If Target.Cells.Count > 1 Then Exit Sub
    県市シート切替 CStr(Target.Value)
End Sub
```

### 「切替中断」が出る

- 県／市シート一覧に登録した名前が現状の可視シートと噛み合わず、切り替えると 1 枚も表示シートが残らなくなるケース
- 表示された案内通り、`Alt+F8` → **`設定チェック`** を実行すると、原因のシート名一覧が出ます
- 表示された不整合（タイポ・「(2)」の付け忘れなど）を `県シート一覧` / `市シート一覧` で直してください

### 「切替エラー」が出る

シート切替中に予期せぬエラーが発生した場合に出ます。

- Excel を一度閉じて開き直して、もう一度試してください
- それでも繰り返し出る場合は、`設定チェック` を実行して問題が無いか確認してください

## カスタマイズ

`県市切替マクロ.bas` の冒頭付近を編集してください。

### 入力シートを変える

```vb
Private Const 入力シート名 As String = "施工計画書（はじめに記入）"
```

対象ブックで使う入力シート名を変えた場合や、別シートで運用したい場合はここを書き換えます。

### 入力セルを変える

```vb
Private Const 入力セル As String = "D3"
```

D3 以外のセルをトリガーにしたい場合はここを変更。変更後は **再度 `初期設定` を実行**してドロップダウンを付け直してください（古い D3 の入力規則は手動で削除）。

### 表示／非表示の対象シートを変える

それぞれ Array() の中をシート名で増減します。**名前はタブ表示と完全一致**必須。

```vb
Public Function 県シート一覧() As Variant
    県シート一覧 = Array( _
        "目次（施工計画書.県土木）", _
        "10（9）.安全教育予定表", _
        ...
    )
End Function

Public Function 市シート一覧() As Variant
    市シート一覧 = Array( _
        "目次（施工計画.市役所）", _
        "10（9）.安全教育予定表 (2)", _
        ...
    )
End Function
```

> 存在しないシート名は無視されるので、安全に試行錯誤できます。
